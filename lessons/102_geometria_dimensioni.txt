# Lezione 102: Dimensioni e Misure
# Dominio: Percezione visiva - dimensioni
# Ancore da: lezioni 00-101

# Dimensioni generali
grande: vedere molto forte / piccolo poco
piccolo: vedere poco debole / grande molto
medio: vedere / grande piccolo

# Lunghezza
lungo: vedere molto / corto poco
corto: vedere poco / lungo molto

# Altezza
alto: vedere sopra / basso sotto
basso: vedere sotto / alto sopra

# Larghezza
largo: vedere molto / stretto poco
stretto: vedere poco / largo molto

# Spessore
spesso: vedere forte / sottile debole
sottile: vedere debole / spesso forte
