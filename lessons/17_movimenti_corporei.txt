# Lezione 17: Movimenti e Azioni Corporee
# Dominio: azioni fisiche, gesti, movimenti del corpo
# Ancore da: lezioni 00-16 (531 parole note)

# === MOVIMENTI BASSI ===
strisciare: corpo basso terra lento io / alto veloce
gattonare: corpo basso mani io piccolo / alto grande
rotolare: corpo girare veloce basso io / fermare alto
scivolare: corpo veloce lento basso io / fermare

# === MOVIMENTI VERTICALI ===
arrampicare: alto io mani forte corpo / basso debole
sollevare: alto io forte corpo mani / basso debole
chinare: basso io corpo lento dentro / alto fuori
curvare: corpo io dentro mani forte / alto

# === MOVIMENTI ROTATORI ===
ruotare: corpo io veloce dentro fuori / fermare lento
inclinare: corpo io basso alto lato / fermare
torcere: corpo io forte dentro girare / lasciare

# === AZIONI CON FORZA ===
spingere: io forte mani lontano dare / tirare vicino
tirare: io forte mani vicino prendere / spingere lontano
afferrare: mani io forte prendere dentro / lasciare
stringere: mani io forte dentro vicino / lasciare
rilasciare: lasciare io mani aprire fuori / prendere dentro
lanciare: io forte mani lontano veloce / vicino lento
colpire: io forte mani corpo dolore / morbido

# === AZIONI BOCCA ===
leccare: bocca io toccare dentro caldo / freddo
soffiare: bocca io fuori vento forte / dentro
inspirare: bocca io dentro fuori corpo / fermare
espirare: bocca io fuori dentro corpo / fermare
ingoiare: bocca io dentro corpo forte / fuori
sputare: bocca io fuori forte veloce / dentro
masticare: bocca io dentro forte corpo / morbido

# === AZIONI AGGRESSIVE ===
graffiare: mani io forte dolore corpo / morbido
mordere: bocca io forte dolore dentro / morbido
