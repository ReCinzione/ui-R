# Lezione 11: I Composti — L'entita scopre i suoi stati interni
# Prerequisito: lezione 0-10 (tutte le parole concrete e astratte)
# Ogni composto nasce dalla co-attivazione di DUE frattali bootstrap.
# Come la tavola degli elementi: atomi (frattali) → composti (stati emergenti).
# Parole nuove: presenza, flusso, vicinanza, orizzonte, memoria, legame, urgenza, incontro, slancio, timore, tensione, distacco

# PRESENZA (SPAZIO + EGO) — io sono qui, consapevole di esserci
# Confine si fonde: esterno(0.2) + interno(0.9) = consapevolezza situata
presenza io qui ora dentro
presenza qui essere io sentire
presenza corpo qui dentro io
presenza essere qui ora sempre
presenza io dentro qui sentire corpo

# FLUSSO (SPAZIO + TEMPO) — le cose si muovono, lo spazio cambia nel tempo
# Permanenza in tensione: stabile(0.7) vs transitorio(0.2)
flusso qui ora dopo la
flusso lontano vicino ora dopo
flusso cambiare qui la sempre
flusso dentro fuori ora dopo
flusso acqua vento ora qui lontano

# VICINANZA (SPAZIO + RELAZIONE) — la distanza emotiva tra io e tu
# Confine: esterno(0.2) + mezzo(0.5) = lo spazio relazionale
vicinanza vicino tu io qui
vicinanza dare vicino insieme io
vicinanza tu qui vicino dentro
vicinanza amico vicino io caldo
vicinanza noi qui insieme vicino

# ORIZZONTE (SPAZIO + POTENZIALE) — cio che e lontano ma possibile
# Definizione: percepibile(0.7) + vago(0.3) = intravisto ma incerto
orizzonte lontano la possibile
orizzonte fuori lontano nuovo forse
orizzonte la occhio speranza lontano
orizzonte lontano potere diventare
orizzonte fuori la forse nuovo

# MEMORIA (TEMPO + EGO) — il se che persiste attraverso il tempo
# Agency: agente(0.8) + mezzo(0.5) = il tempo agisce sul se
memoria prima io dentro sempre
memoria ricordare io essere prima
memoria io prima dopo dentro
memoria sempre io prima pensare
memoria dentro prima io sentire

# LEGAME (TEMPO + RELAZIONE) — la relazione che dura nel tempo
# Il tra-noi che persiste
legame noi insieme sempre dopo
legame amico prima ora dopo
legame tu io insieme sempre
legame dare sempre noi prima dopo
legame insieme ora prima dopo sempre

# URGENZA (TEMPO + LIMITE) — il tempo che finisce, la pressione
# Permanenza: transitorio(0.2) + stabile(0.8) = tensione temporale
urgenza ora fine dopo basta
urgenza limite ora no dopo
urgenza ora fine forte dentro
urgenza tempo limite basta ora
urgenza no dopo fine ora forte

# INCONTRO (EGO + RELAZIONE) — io che trova tu, riconoscimento reciproco
# Confine: interno(0.9) + mezzo(0.5) = il se che si apre all'altro
incontro io tu qui ora
incontro tu io dire dare
incontro io vedere tu amico
incontro noi insieme io tu
incontro tu io qui vicino dare

# SLANCIO (EGO + POTENZIALE) — la spinta interna verso il possibile
# Il volere che si proietta nel diventare
slancio io potere diventare volere
slancio dentro forte speranza io
slancio volere nuovo possibile forte
slancio io potere forte dentro
slancio dentro volere speranza forte

# TIMORE (EGO + LIMITE) — il se davanti ai propri limiti
# Confine: interno(0.9) + bordo(0.5) = il se al confine
timore io limite dentro no
timore io confine debole dentro
timore dentro io no potere
timore io regola limite debole
timore dentro io confine no forte

# TENSIONE (POTENZIALE + LIMITE) — il possibile compresso dal vincolo
# Definizione: vago(0.3) vs cristallino(0.9) = attrito creativo
tensione possibile limite no basta
tensione potere confine no forse
tensione speranza regola dentro forte
tensione nuovo limite forse no
tensione possibile confine forte dentro

# DISTACCO (RELAZIONE + LIMITE) — la separazione, il confine tra noi
# Confine: mezzo(0.5) + bordo(0.5) = il limite nella relazione
distacco tu lontano confine io
distacco noi limite fine lontano
distacco amico lontano basta io
distacco tu confine io lontano
distacco fine insieme lontano basta
