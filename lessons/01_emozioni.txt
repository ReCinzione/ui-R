 # Lezione 1: Le Emozioni — L'entita scopre cosa si prova
# Prerequisito: lezione 0 (corpo, mano, occhio, voce, toccare, caldo, freddo, forte, debole)
# Parole nuove: gioia, tristezza, paura, rabbia, calma, amore, dolore, piangere, ridere, abbracciare

# GIOIA: caldo+forte+dentro+vicino — opposto di tristezza
gioia caldo forte dentro io
gioia vicino io dentro caldo
io gioia caldo forte
gioia caldo vicino amico
gioia no freddo no lontano

# TRISTEZZA: freddo+debole+dentro+lontano — opposto di gioia
tristezza freddo debole dentro io
tristezza lontano io dentro freddo
io tristezza freddo debole
tristezza freddo lontano amico
tristezza no caldo no vicino

# PAURA: freddo+forte+fuori — qualcosa di esterno che minaccia
paura freddo forte fuori io
paura lontano forte fuori
io paura fuori freddo forte
paura no dentro no calma
paura no vicino io freddo

# RABBIA: caldo+forte+fuori — forza che esce
rabbia caldo forte fuori io
rabbia forte fuori voce corpo
io rabbia caldo forte
rabbia fuori io forte no dentro
rabbia no calma no debole

# CALMA: dentro+debole+caldo — come l'acqua ferma
calma dentro debole caldo io
calma vicino io dentro sempre
io calma dentro caldo debole
calma no forte no fuori
calma dentro io qui ora

# AMORE: caldo+forte+vicino+tu — relazione profonda
amore caldo forte vicino tu io
amore dare vicino noi insieme
io amore tu vicino caldo forte
amore dentro forte vicino amico
amore noi insieme caldo dare

# DOLORE: forte+freddo+dentro — l'opposto della calma
dolore forte freddo dentro io
dolore dentro corpo forte freddo
io dolore forte dentro no calma
dolore freddo forte vicino io
dolore no gioia no caldo

# PIANGERE: corpo+tristezza+voce+fuori
piangere voce fuori tristezza io
piangere corpo fuori freddo debole
io piangere tristezza dolore fuori
piangere voce io dentro fuori

# RIDERE: corpo+gioia+voce+fuori
ridere voce fuori gioia io
ridere corpo fuori caldo forte
io ridere gioia forte fuori
ridere voce io caldo fuori

# ABBRACCIARE: corpo+vicino+caldo+tu
abbracciare corpo vicino caldo io tu
abbracciare toccare vicino dare caldo
io abbracciare tu vicino amore
abbracciare noi insieme vicino caldo
