# Lezione 49: Acqua e Liquidi
# Dominio: fluidi, stati liquidi, idraulica
# Ancore da: lezioni 00-48 (1055+ parole note)

# === MOVIMENTI ACQUA ===
scorrere: acqua io veloce lungo dentro / fermo
fluire: acqua io lento lungo dentro / fermo
sgorgare: acqua io veloce fuori dentro / dentro
zampillare: acqua io veloce alto fuori / basso
gocciolare: acqua io piccolo lento cadere / veloce
stillare: acqua io piccolo lento cadere / veloce

# === STATI ACQUA ===
bagnato: acqua io dentro dentro morbido / secco
umido: acqua io dentro piccolo dentro / secco
fradicio: acqua io dentro grande dentro / secco
inzuppato: acqua io dentro grande dentro / secco
asciutto: io dentro no acqua / bagnato
secco: io dentro no acqua / bagnato

# === AZIONI CON ACQUA ===
bagnare: acqua io dentro fare dentro / asciugare
inumidire: acqua io dentro piccolo fare / secco
immergere: acqua io dentro dentro dentro / fuori
sommergere: acqua io dentro dentro grande / fuori
affogare: acqua io dentro dentro no / fuori
annegare: acqua io dentro dentro morire / vivere

# === PULIZIA ===
lavare: acqua io dentro fare pulire / sporco
sciacquare: acqua io dentro veloce fare / sporco
detergere: acqua io dentro forte pulire / sporco
mondare: acqua io dentro forte pulire / sporco

# === VERSARE ===
versare: acqua io dentro fuori fare / dentro
rovesciare: acqua io veloce fuori dentro / dentro
traboccare: acqua io fuori grande dentro / dentro
spargere: acqua io fuori grande dentro / dentro

# === ASSORBIRE ===
assorbire: acqua io dentro dentro prendere / dare
imbevere: acqua io dentro dentro dentro / fuori
impregnarsi: acqua io dentro dentro dentro / no
