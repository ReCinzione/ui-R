# Lezione 7: Le Qualita — L'entita scopre come sono le cose
# Prerequisito: lezione 0-6
# Parole nuove: bello, brutto, buono, cattivo, grande, piccolo, alto, basso, lungo, corto, duro, morbido, dolce, amaro, chiaro, scuro, veloce, lento, pesante, leggero

# BELLO: caldo+gioia+dentro+occhio
bello caldo gioia dentro io
bello occhio io caldo forte
bello dentro gioia io
bello no brutto caldo

# BRUTTO: freddo+no+gioia
brutto freddo no gioia io
brutto occhio io freddo dentro
brutto no bello freddo
brutto fuori freddo io

# BUONO: caldo+dare+vicino
buono caldo dare vicino io
buono vicino io dare caldo
buono dentro caldo dare
buono no cattivo

# CATTIVO: freddo+no+dare
cattivo freddo no dare io
cattivo lontano io no dare
cattivo freddo fuori no buono
cattivo no buono freddo

# GRANDE: forte+lontano+confine
grande forte lontano confine fuori
grande no piccolo forte
grande io fuori forte lontano
grande forte confine lontano

# PICCOLO: debole+vicino+confine
piccolo debole vicino confine dentro
piccolo no grande debole
piccolo io dentro vicino debole
piccolo dentro confine vicino

# ALTO: fuori+lontano+cielo
alto fuori lontano cielo grande
alto la cielo io fuori
alto grande fuori lontano
alto no basso cielo

# BASSO: fuori+vicino+terra
basso fuori vicino terra piccolo
basso qui terra io vicino
basso piccolo vicino terra
basso no alto terra

# LUNGO: lontano+dopo+grande
lungo lontano dopo io grande
lungo qui la dopo lontano
lungo grande dopo lontano
lungo no corto

# CORTO: vicino+ora+piccolo
corto vicino ora io piccolo
corto qui ora piccolo vicino
corto no lungo piccolo
corto ora vicino qui

# DURO: forte+limite+no+cambiare
duro forte limite no cambiare
duro forte dentro no debole
duro toccare forte limite
duro no morbido forte

# MORBIDO: debole+no+limite+toccare
morbido debole no limite toccare
morbido toccare caldo debole
morbido no duro debole
morbido dentro debole vicino

# DOLCE: caldo+buono+dentro
dolce caldo buono dentro io
dolce dentro caldo buono toccare
dolce io dentro caldo
dolce no amaro caldo

# AMARO: freddo+dentro+no+buono
amaro freddo dentro no buono io
amaro freddo dentro forte
amaro no dolce freddo
amaro dentro freddo io

# CHIARO: luce+forte+fuori
chiaro luce forte fuori occhio
chiaro luce io forte
chiaro no scuro luce
chiaro fuori forte luce

# SCURO: buio+debole+dentro
scuro buio debole dentro io
scuro no chiaro buio
scuro dentro buio freddo
scuro no luce dentro

# VELOCE: forte+ora+cambiare
veloce forte ora cambiare io
veloce correre forte ora
veloce no lento forte ora
veloce fare forte ora

# LENTO: debole+dopo+cambiare
lento debole dopo cambiare io
lento debole dopo forse
lento no veloce debole
lento fare debole dopo

# PESANTE: forte+dentro+basso
pesante forte dentro basso io
pesante forte basso terra
pesante no leggero forte
pesante dentro forte corpo

# LEGGERO: debole+dentro+alto
leggero debole dentro alto io
leggero debole alto cielo
leggero no pesante debole
leggero dentro debole fuori
