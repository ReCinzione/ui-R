# Lezione 100: Forme Geometriche Base
# Dominio: Percezione visiva - forme
# Ancore da: lezioni 00-99

# Forme base
cerchio: forma vedere rotondo / quadrato angolo
quadrato: forma vedere angolo / cerchio rotondo
rettangolo: forma vedere angolo lungo / cerchio rotondo
triangolo: forma vedere angolo tre / cerchio rotondo
linea: forma vedere lungo sottile / largo
punto: forma vedere piccolo / grande
forma: vedere essere / niente
