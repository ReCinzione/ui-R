# Lezione 18: Qualità e Attributi Sensoriali
# Dominio: qualità fisiche, sensazioni, attributi percettivi
# Ancore da: lezioni 00-17 (559 parole note)

# === TEMPERATURA E SENSAZIONI ===
tiepido: caldo freddo dentro io sentire / forte
gelido: freddo forte io dentro corpo / caldo
bollente: caldo forte io dentro dolore / freddo
fresco: freddo leggero io dentro bene / caldo

# === CONSISTENZA ===
ruvido: duro forte toccare io corpo / morbido
liscio: morbido toccare io dentro buono / ruvido duro
appiccicoso: toccare io dentro forte vicino / lasciare
scivoloso: toccare io veloce dentro morbido / fermare duro
viscoso: dentro io toccare lento forte / veloce

# === PESO E DENSITÀ ===
massiccio: pesante grande forte io dentro / leggero piccolo
denso: dentro forte pesante io vicino / leggero
sottile: piccolo leggero dentro io debole / grande forte
spesso: grande forte dentro io pesante / piccolo leggero

# === FORMA ===
aguzzo: forte dentro io dolore toccare / morbido
appuntito: forte piccolo dentro io dolore / morbido
tondo: dentro io morbido toccare buono / duro
piatto: basso dentro io grande toccare / alto piccolo

# === LUMINOSITÀ ===
brillante: luce forte io dentro bello / buio
opaco: buio dentro io debole luce / forte
lucente: luce forte io dentro bello / buio
scintillante: luce forte veloce io dentro / buio lento

# === SAPORI ===
salato: dentro io bocca forte sentire / dolce
acido: dentro io bocca forte dolore / dolce
aspro: dentro io bocca forte cattivo / dolce buono
insipido: dentro io bocca debole no / forte

# === ODORI ===
fragrante: dentro io sentire buono caldo / cattivo
puzzolente: dentro io sentire cattivo forte / buono
