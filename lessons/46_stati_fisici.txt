# Lezione 46: Stati Fisici e Condizioni
# Dominio: condizioni corporee, salute, malattia
# Ancore da: lezioni 00-45 (1055 parole note)

# === SALUTE ===
sano: corpo io forte buono dentro / malato
vigoroso: corpo io forte grande dentro / debole
robusto: corpo io forte duro dentro / debole
florido: corpo io forte buono dentro / debole

# === MALATTIA ===
malato: corpo io debole cattivo dentro / sano
infermo: corpo io debole dentro dolore / sano
ammalato: corpo io debole cattivo dentro / sano
sofferente: corpo io dentro dolore forte / sano

# === STANCHEZZA ===
stanco: corpo io debole dentro lento / forte
affaticato: corpo io debole dentro lento / forte
esausto: corpo io debole grande dentro / forte
spossato: corpo io debole grande dentro / forte
fiacco: corpo io debole dentro lento / forte

# === ENERGIA ===
energico: corpo io forte veloce dentro / debole
vivace: corpo io forte veloce dentro / lento
arzillo: corpo io forte veloce dentro / debole
languido: corpo io debole lento dentro / forte
fiacco: corpo io debole lento dentro / forte

# === DOLORE ===
dolorante: corpo io dentro dolore forte / no
indolenzito: corpo io dentro dolore lento / no
sofferente: corpo io dentro dolore forte / no

# === FEBBRE ===
febbrile: corpo io dentro caldo forte / freddo
ardente: corpo io dentro caldo forte / freddo
accaldato: corpo io dentro caldo dentro / freddo
infreddolito: corpo io dentro freddo forte / caldo
