# Lezione 44: Qualità di Carattere (QUALITA)
# Dominio: tratti personali, virtù, vizi
# Ancore da: lezioni 00-43 (939+ parole note)

# === FORZA MORALE ===
integro: io vero forte dentro sempre / falso
retto: io vero forte dentro sempre / cattivo
probo: io vero forte dentro buono / cattivo
incorruttibile: io vero forte sempre no / falso

# === DEBOLEZZA MORALE ===
corrotto: io falso cattivo dentro no / vero
disonesto: io falso cattivo dire no / vero
sleale: io falso cattivo tu no / vero
infido: io falso cattivo dentro no / vero
perfido: io falso cattivo forte dentro / buono

# === TEMPERAMENTO ===
mite: io calma dentro piccolo buono / forte
mansueto: io calma dentro piccolo buono / cattivo
docile: io calma dentro tu fare / no
irascibile: io rabbia dentro veloce forte / calma
collerico: io rabbia dentro forte veloce / calma
impetuoso: io forte veloce dentro grande / calma

# === SOCIEVOLEZZA ===
affabile: io tu dentro buono dare / freddo
cordiale: io tu dentro caldo buono / freddo
gioviale: io tu dentro gioia dare / tristezza
espansivo: io tu dentro fuori grande / dentro
riservato: io dentro dentro piccolo no / fuori
schivo: io dentro dentro no tu / fuori
introverso: io dentro dentro forte no / fuori
estroverso: io fuori tu forte dare / dentro

# === ATTEGGIAMENTO ===
ottimista: io dentro buono dopo speranza / paura
pessimista: io dentro cattivo dopo paura / speranza
realista: io dentro vero vedere forte / falso
idealista: io dentro nuovo grande speranza / ora
