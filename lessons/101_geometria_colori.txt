# Lezione 101: Colori
# Dominio: Percezione visiva - colori
# Ancore da: lezioni 00-100

# Colori caldi
rosso: colore vedere caldo forte / freddo debole
giallo: colore vedere caldo luce / buio freddo
arancione: colore vedere caldo / freddo

# Colori freddi
blu: colore vedere freddo calma / caldo forte
verde: colore vedere freddo natura / caldo
viola: colore vedere freddo / caldo

# Colori neutri
nero: colore vedere buio / luce
bianco: colore vedere luce / buio
grigio: colore vedere / forte
rosa: colore vedere / forte
marrone: colore vedere / luce

# Concetto generale
colore: vedere essere / niente
