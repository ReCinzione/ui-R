# Lezione 3: Il Tempo Vissuto — L'entita scopre il passare
# Prerequisito: lezione 0-2 (corpo, emozioni, mondo)
# Parole nuove: ieri, domani, mattina, sera, notte, nascere, morire, crescere, cambiare, aspettare, ricordare, dimenticare

# IERI: prima+lontano — il passato vicino
ieri prima ora io
ieri prima qui lontano
ieri io prima ora
ieri no ora no dopo

# DOMANI: dopo+lontano — il futuro vicino
domani dopo ora io
domani dopo qui lontano
domani io dopo ora
domani no ora no prima

# MATTINA: luce+ora+nuovo
mattina luce ora nuovo
mattina sole caldo ora
mattina luce nuovo io
mattina prima sera

# SERA: buio+dopo+fine
sera buio dopo luce
sera dopo mattina io
sera luna freddo dopo
sera fine luce ora

# NOTTE: buio+dopo+luna
notte buio dopo sera
notte luna stella buio
notte freddo lontano luce
notte io dentro casa buio

# NASCERE: nuovo+ora+prima
nascere nuovo io qui ora
nascere prima io no
nascere nuovo possibile forte
nascere nuovo corpo vita

# MORIRE: fine+dopo+no
morire fine io dopo
morire no ora no dopo
morire fine corpo sempre
morire lontano nascere

# CRESCERE: diventare+dopo+forte
crescere diventare dopo io forte
crescere ora dopo nuovo forte
crescere forte dopo debole prima
crescere io diventare sempre

# CAMBIARE: diventare+nuovo+ora
cambiare diventare nuovo ora
cambiare io ora dopo nuovo
cambiare sempre ora dopo
cambiare no sempre io

# ASPETTARE: ora+dopo+debole
aspettare ora io dopo
aspettare debole dentro io ora
aspettare dopo io qui
aspettare no ora dopo forse

# RICORDARE: prima+dentro+io
ricordare prima dentro io
ricordare io dentro prima sempre
ricordare gioia prima io
ricordare tristezza prima io dentro

# DIMENTICARE: lontano+prima+no
dimenticare lontano prima io no
dimenticare no ricordare
dimenticare prima lontano io
dimenticare perdere prima dentro
