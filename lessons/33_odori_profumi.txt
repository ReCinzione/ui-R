# Lezione 33: Odori e Profumi
# Dominio: percezione olfattiva, naso
# Ancore da: lezioni 00-32 (756+ parole note)

# === ODORI PIACEVOLI ===
profumo: io dentro buono forte fiore / cattivo
fragranza: io dentro buono piccolo fiore / cattivo
aroma: io dentro buono forte caldo / freddo
essenza: io dentro buono piccolo dentro / fuori

# === ODORI SGRADEVOLI ===
puzzo: io dentro cattivo forte no / buono
fetore: io dentro cattivo forte grande / buono
tanfo: io dentro cattivo forte grande / buono
lezzo: io dentro cattivo forte dentro / buono

# === QUALITÀ OLFATTIVE ===
profumato: io dentro buono forte fiore / cattivo
puzzolente: io dentro cattivo forte no / buono
fragrante: io dentro buono forte dentro / cattivo
maleodorante: io dentro cattivo forte no / buono
inodore: io dentro no piccolo / forte

# === INTENSITÀ ===
penetrante: io dentro forte grande dentro / debole
intenso: io dentro forte grande dentro / debole
delicato: io dentro piccolo debole buono / forte
tenue: io dentro piccolo debole dentro / forte
