# Lezione 38: Processi Cognitivi (PENSIERO)
# Dominio: operazioni mentali, cognizione
# Ancore da: lezioni 00-37 (846+ parole note)

# === RIFLESSIONE ===
contemplare: pensare io dentro lento calma / veloce
ponderare: pensare io dentro lento forte / veloce
considerare: pensare io dentro forte sapere / no

# === COMPRENSIONE ===
intuire: sapere io dentro veloce no / lento
afferrare: capire io dentro veloce forte / no
cogliere: capire io dentro veloce forte / no
percepire: sentire io dentro sapere dentro / no

# === IMMAGINAZIONE ===
concepire: pensare io dentro nuovo forte / vecchio
ideare: pensare io dentro nuovo fare / vecchio
inventare: pensare io dentro nuovo fare / vecchio

# === ANALISI ===
analizzare: pensare io dentro forte piccolo / grande
dedurre: pensare io dentro forte dopo / prima
indurre: pensare io dentro forte grande / piccolo
inferire: pensare io dentro forte sapere / no
scrutare: vedere io dentro forte piccolo / grande
esaminare: vedere io dentro forte pensare / no

# === SINTESI ===
sintetizzare: pensare io dentro grande insieme / piccolo
unificare: pensare io dentro insieme forte / separare
