# Lezione 26: Qualità Morali e Virtù
# Dominio: etica, carattere, virtù
# Ancore da: lezioni 00-25 (787 parole note)

# === VIRTÙ POSITIVE ===
onesto: io vero dire dare rispetto / falso
sincero: io vero dire dentro fuori / falso
leale: io tu sempre dare rispetto / tradire
fedele: io tu sempre insieme dare / lasciare
coraggioso: io forte paura fare / debole
valoroso: io forte paura grande fare / debole

# === GENEROSITÀ ===
generoso: io dare tu grande caldo / prendere
altruista: io dare tu sempre no / io
disinteressato: io dare tu no volere / prendere
avaro: io prendere no dare freddo / dare
egoista: io prendere no tu dare / noi
ingordo: io prendere grande sempre no / dare

# === SAGGEZZA ===
saggio: io sapere pensare forte vecchio / giovane
prudente: io pensare prima fare lento / veloce
accorto: io pensare vedere forte prima / dopo
avveduto: io pensare vedere forte dentro / fuori
stolto: io no pensare fare cattivo / saggio
sciocco: io no pensare fare debole / saggio

# === UMILTÀ ===
umile: io piccolo dentro calma rispetto / grande
modesto: io piccolo dire no grande / grande
superbo: io grande dire forte alto / piccolo
arrogante: io grande dire forte cattivo / umile
vanitoso: io grande vedere bello no / brutto

# === TEMPERANZA ===
temperante: io forte dentro fermare limite / no
sobrio: io forte dentro fermare bocca / no
moderato: io forte dentro piccolo limite / grande
dissoluto: io debole no fermare grande / forte
