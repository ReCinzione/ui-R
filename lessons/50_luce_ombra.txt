# Lezione 50: Luce e Ombra
# Dominio: illuminazione, visibilità, contrasti
# Ancore da: lezioni 00-49 (1055+ parole note)

# === LUCE ===
illuminare: luce io dentro fare vedere / buio
rischiarare: luce io dentro fare vedere / buio
risplendere: luce io forte grande dentro / buio
rifulgere: luce io forte grande dentro / buio
irradiare: luce io fuori forte dentro / dentro
emanare: luce io fuori dentro dentro / no

# === RIFLESSI ===
riflettere: luce io dentro vedere dentro / no
specchiare: luce io dentro vedere dentro / no
riverberare: luce io dentro forte vedere / no
baluginare: luce io piccolo veloce dentro / no

# === OMBRA ===
ombreggiare: buio io dentro fare dentro / luce
adombrare: buio io dentro fare dentro / luce
oscurare: buio io dentro forte fare / luce
offuscare: buio io dentro dentro fare / chiaro
annebbiare: buio io dentro dentro dentro / chiaro
velare: buio io dentro dentro piccolo / chiaro

# === TRASPARENZA ===
trasparente: io vedere dentro dentro luce / buio
limpido: io vedere dentro chiaro dentro / buio
cristallino: io vedere dentro chiaro forte / buio
opaco: io vedere no dentro buio / chiaro
torbido: io vedere no dentro buio / chiaro

# === VISIBILITÀ ===
visibile: io vedere dentro forte chiaro / no
invisibile: io vedere no dentro no / vedere
apparire: io vedere dentro nuovo forte / no
scomparire: io vedere no lontano dentro / vedere
manifestarsi: io vedere dentro forte nuovo / nascosto
celarsi: io vedere no dentro dentro / vedere
nascondersi: io vedere no dentro dentro / vedere
