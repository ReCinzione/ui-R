# Lezione 103: Relazioni Spaziali
# Dominio: Percezione visiva - spazio
# Ancore da: lezioni 00-102

# Posizione base
qui: essere vedere / la
la: essere vedere / qui

# Verticalità
sopra: vedere alto / sotto basso
sotto: vedere basso / sopra alto

# Profondità
dentro: essere / fuori
fuori: essere / dentro

# Distanza
vicino: essere / lontano
lontano: essere / vicino

# Parti
centro: essere dentro / bordo fuori
bordo: essere fuori / centro dentro

# Lateralità
destra: essere / sinistra
sinistra: essere / destra
