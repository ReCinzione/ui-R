# Lezione 88: Calibrazione EMOZIONE
# Dominio: stati emotivi puri, intensità, valenza
# Ancore da: lezioni 00-87 (2024 parole note)

# === EMOZIONI PURE ===
emozione: io sentire forte caldo dentro / no
sentimento: io sentire profondo lungo tempo / no
affetto: io tu sentire caldo dare / freddo
stato: io essere dentro ora qui / no

# === INTENSITÀ EMOTIVA ===
intenso: io sentire forte grande caldo / debole
profondo: io sentire dentro forte lungo / fuori
superficiale: io sentire fuori piccolo veloce / profondo
lieve: io sentire piccolo debole calma / forte

# === VALENZA ===
piacevole: io sentire buono caldo dentro / cattivo
spiacevole: io sentire cattivo freddo dentro / buono
neutro: io sentire no forte calma / forte
ambivalente: io sentire insieme opposto dentro / no

# === TRANSIZIONI ===
commuovere: io sentire forte nuovo caldo / no
turbare: io sentire cattivo forte paura / calma
rasserenare: io sentire buono calma luce / cattivo
eccitare: io sentire forte veloce caldo / calma
placare: io sentire calma lento dentro / forte

# === ESPRESSIONE ===
esprimere: io fuori fare dire vedere / no
manifestare: io fuori fare vedere forte / nascondere
reprimere: io dentro forte no fuori / esprimere
contenere: io dentro forte fermare dentro / fuori
sfogare: io fuori forte veloce dentro / contenere
