# Lezione 41: Azioni Sociali (SOCIETA + AZIONE)
# Dominio: interazioni, comportamenti collettivi
# Ancore da: lezioni 00-40 (939 parole note)

# === COOPERAZIONE ===
collaborare: io tu insieme fare forte / solo
cooperare: io tu insieme fare dare / solo
condividere: io tu dare insieme dentro / prendere
partecipare: io noi insieme fare dentro / fuori
contribuire: io noi dare fare insieme / prendere

# === CONFLITTO ===
litigare: io tu no insieme rabbia / calma
contendere: io tu no dare prendere / dare
disputare: io tu no insieme dire / calma
combattere: io tu no forte paura / calma
scontrarsi: io tu no forte insieme / calma

# === LEADERSHIP ===
comandare: io tu dire forte fare / no
guidare: io tu dare fare insieme / solo
dirigere: io tu dire fare forte / no
governare: io noi dire forte grande / piccolo
dominare: io tu forte grande no / debole

# === SOTTOMISSIONE ===
obbedire: io tu fare dire no / no
sottomettersi: io tu dentro debole no / forte
servire: io tu dare fare dentro / prendere
seguire: io tu dopo insieme fare / prima

# === MEDIAZIONE ===
negoziare: io tu dire insieme dare / no
mediare: io tu noi insieme dare / no
conciliare: io tu insieme calma dare / no
arbitrare: io tu noi dire forte / no
