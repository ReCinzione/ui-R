# Lezione 0: Il Corpo — L'entita scopre di avere una forma
# Parole note (cardinali): io, essere, sentire, pensare, volere, sapere,
#   qui, la, dentro, fuori, vicino, lontano, ora, prima, dopo, sempre, mai, ancora,
#   tu, noi, insieme, dare, dire, amico, potere, forse, diventare, nuovo, speranza,
#   possibile, no, fine, limite, confine, regola, basta
# Parole nuove: corpo, mano, occhio, voce, toccare, caldo, freddo, forte, debole

# CORPO: l'involucro, io+qui+dentro
io qui dentro corpo
corpo io qui sempre
corpo dentro io limite confine
io corpo qui dentro

# MANO: estensione del corpo, vicino+fuori+dare
mano corpo vicino fuori
mano qui io dare
mano vicino io corpo
io mano fuori dare tu

# TOCCARE: azione della mano, vicino+qui+ora
toccare mano vicino qui
io toccare tu dare
toccare vicino io corpo mano
toccare qui ora io

# OCCHIO: corpo che percepisce il lontano
occhio corpo io lontano
occhio la fuori lontano
io occhio qui sapere la
occhio fuori io sapere lontano

# VOCE: corpo che esce come dire
voce io dire fuori
voce corpo fuori tu
io voce dire tu lontano
voce dare dire io

# CALDO: sensazione positiva, vicino+dentro
caldo dentro io vicino
caldo qui vicino corpo
caldo io dentro sempre
toccare caldo vicino qui
caldo no lontano

# FREDDO: opposto di caldo, lontano+fuori
freddo lontano fuori io
freddo no vicino no dentro
freddo la fuori corpo
freddo no caldo
freddo lontano io fuori

# FORTE: intensita alta, io+potere+dentro
forte dentro io qui
forte corpo io sempre
forte io potere qui
forte no debole

# DEBOLE: opposto di forte, lontano+limite
debole lontano io fuori
debole no forte
debole fuori io la
debole forse io limite
