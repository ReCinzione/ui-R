# Lezione 4: Le Relazioni — L'entita scopre gli altri
# Prerequisito: lezione 0-3 (corpo, emozioni, mondo, tempo)
# Parole nuove: madre, padre, figlio, fratello, famiglia, parlare, ascoltare, capire, aiutare, fiducia, rispetto, solitudine

# MADRE: dare+amore+sempre+vicino
madre dare amore io sempre
madre vicino io caldo forte
madre io prima sempre amore
madre dare corpo io nascere

# PADRE: dare+forte+io+sempre
padre dare forte io sempre
padre forte vicino io
padre io prima sempre forte
padre dare io crescere

# FIGLIO: nuovo+nascere+dare — chi viene dopo
figlio nuovo nascere io dare
figlio io dopo nuovo
figlio nascere crescere io
figlio piccolo nuovo dare amore

# FRATELLO: vicino+insieme+io — chi cammina accanto
fratello vicino insieme io sempre
fratello io vicino amico
fratello insieme io forte
fratello noi vicino sempre

# FAMIGLIA: noi+insieme+sempre+casa
famiglia noi insieme sempre
famiglia madre padre figlio fratello
famiglia dentro casa caldo
famiglia noi insieme sempre amore

# PARLARE: dire+tu+voce+fuori
parlare dire io tu voce
parlare voce io fuori tu
parlare dire tu io
parlare voce dare tu

# ASCOLTARE: tu+dire+dentro — ricevere voce
ascoltare tu dire dentro io
ascoltare dentro io tu voce
ascoltare io tu dare dire
ascoltare dentro tu parlare

# CAPIRE: dentro+pensare+sapere
capire dentro pensare io tu
capire pensare sapere dentro
capire io tu dentro pensare
capire sapere dopo pensare

# AIUTARE: dare+tu+insieme+forte
aiutare dare tu io forte
aiutare insieme io tu dare
aiutare noi forte insieme
aiutare dare forte tu

# FIDUCIA: caldo+tu+dentro+dare
fiducia caldo vicino tu io
fiducia dare tu dentro caldo
fiducia io tu vicino sempre
fiducia caldo dentro dare tu

# RISPETTO: tu+io+dare+confine
rispetto tu io dare confine
rispetto io tu limite dare
rispetto dare tu io confine
rispetto tu io vicino regola

# SOLITUDINE: io+lontano+tu+freddo
solitudine io lontano tu freddo
solitudine no vicino no insieme
solitudine dentro io freddo lontano
solitudine no amico lontano io
