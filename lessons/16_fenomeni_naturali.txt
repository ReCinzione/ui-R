# Lezione 16: Fenomeni Naturali e Atmosferici
# Dominio: eventi naturali, atmosfera, elementi
# Ancore da: lezioni 00-15 (508 parole note)

# === FENOMENI ATMOSFERICI ===
tuono: forte voce cielo io paura / silenzio calma
fulmine: luce forte veloce cielo io / buio lento
nebbia: buio lento fuori io freddo / luce chiaro sole
brina: freddo acqua mattina piccolo terra / caldo fuoco
rugiada: acqua mattina luce piccolo terra / fuoco sera
ghiaccio: acqua freddo forte duro io / caldo morbido
neve: freddo leggero cadere cielo io / caldo terra
grandine: freddo forte cadere cielo dolore / caldo morbido

# === EVENTI VIOLENTI ===
tempesta: vento forte pioggia cielo io / calma sole
uragano: vento forte grande veloce paura / calma lento
tornado: vento forte veloce alto io / calma basso
terremoto: terra forte io paura dentro / calma fermare
valanga: neve forte veloce basso io / lento alto
frana: terra forte cadere basso io / alto fermare
eruzione: fuoco forte alto terra io / freddo basso

# === ELEMENTI TRASFORMATI ===
lava: fuoco forte terra caldo io / freddo acqua
cenere: fuoco dopo buio leggero io / forte pesante
fumo: fuoco alto leggero buio io / chiaro basso
vapore: acqua caldo alto leggero io / freddo basso

# === ACQUA IN MOVIMENTO ===
onda: acqua forte mare io dentro / calma fermare
marea: acqua mare dopo sempre io / fermare
corrente: acqua forte veloce dentro io / lento fermare
cascata: acqua forte cadere alto io / lento basso
sorgente: acqua nascere terra dentro io / morire

# === LUOGHI NATURALI ===
palude: acqua terra dentro lento io / fuori veloce
deserto: terra caldo grande lontano io / acqua freddo
oasi: acqua dentro deserto albero io / lontano
