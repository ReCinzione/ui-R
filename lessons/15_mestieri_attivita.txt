# Lezione 15: Mestieri e Attività Umane
# Dominio: lavori, professioni, attività pratiche
# Ancore da: lezioni 00-14 (482 parole note)

# === LAVORI DELLA TERRA ===
agricoltore: terra seme crescere io fare / mare acqua
pastore: animale terra io dare caldo / bosco lontano
pescatore: mare acqua pesce io prendere / terra fuori
cacciatore: animale bosco io forte prendere / casa dentro
seminatore: seme terra io dare mani / prendere

# === ARTIGIANI ===
fabbro: fuoco forte pietra io fare / freddo acqua
falegname: albero io fare mani forte / pietra duro
vasaio: terra acqua io fare mani / fuoco forte
tessitore: mani io fare dentro lungo / rompere corto
sarto: mani io fare corpo dare / rompere

# === COSTRUTTORI ===
muratore: pietra casa io fare forte / rompere debole
carpentiere: albero casa io fare mani / pietra terra
architetto: casa pensare io creare grande / piccolo
scultore: pietra io fare bello mani / brutto rompere

# === COMMERCIO ===
venditore: dare io tu parlare fuori / prendere dentro
compratore: prendere io dare tu volere / lasciare
negoziante: dare prendere io tu casa / lontano

# === CURA E SERVIZIO ===
guaritore: corpo io dare caldo forte / dolore freddo
cuoco: fuoco io fare dare caldo / freddo
servitore: io dare tu fare sempre / prendere
infermiere: corpo io dare aiutare dentro / fuori
levatrice: nascere io dare aiutare corpo / morire

# === SAPERE ===
allievo: imparare io sapere nuovo dentro / insegnare
scrittore: dire io pensare creare mani / silenzio
poeta: dire io bello dentro creare / brutto cattivo
filosofo: pensare io dentro sapere domanda / fare fuori
