# Lezione 59: Tempo Concreto e Avverbi Temporali
# Dominio: riferimenti temporali specifici, durate, frequenze
# Ancore da: lezioni 00-58 (1312+ parole note)

# === RIFERIMENTI TEMPORALI ===
oggi: tempo ora io qui dentro / ieri
stanotte: tempo notte ora io / giorno
adesso: tempo ora io qui dentro / dopo
subito: tempo ora veloce io / dopo
presto: tempo prima veloce io / tardi
tardi: tempo dopo lento io / presto
allora: tempo prima io dentro / ora

# === DURATE ===
mentre: tempo io dentro insieme fare / dopo
durante: tempo io dentro dentro fare / fuori
fino: tempo io dentro limite dopo / prima
quando: tempo io dentro domanda dentro / no
poi: tempo dopo io fare dentro / prima
infine: tempo dopo fine io / prima
intanto: tempo io dentro insieme fare / dopo

# === FREQUENZE ===
spesso: tempo io dentro sempre forte / mai
raramente: tempo io dentro piccolo mai / sempre
talvolta: tempo io dentro piccolo forse / sempre
ogni: tempo io dentro sempre dentro / no
sovente: tempo io dentro forte sempre / mai

# === CONTINUITÀ ===
ancora: tempo io dentro sempre dentro / no
tuttora: tempo io dentro ora sempre / no
sempre: tempo io dentro dentro dentro / mai
mai: tempo io no dentro dentro / sempre

# === SEQUENZE ===
prima: tempo io dentro dentro prima / dopo
dopo: tempo io dentro dentro dopo / prima
infine: tempo io dentro fine dopo / prima
dapprima: tempo io dentro prima prima / dopo
