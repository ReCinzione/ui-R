# GRAMMATICA ITALIANA - Libro Base per Prometeo
# Obiettivo: Insegnare strutture sintattiche attraverso pattern topologici
# Approccio: Non regole astratte, ma pattern ricorrenti nel campo

## CAPITOLO 1: LA FRASE SEMPLICE

# Soggetto + Verbo
io cammino
tu corri
lui parla
noi mangiamo
voi dormite
loro pensano

# Soggetto + Verbo + Complemento Oggetto

tu bevi l'acqua
lui legge il libro
noi vediamo il cielo
voi sentite il suono
loro toccano la pietra

# Soggetto + Verbo + Aggettivo
io sono felice
tu sei triste
lui è forte
noi siamo calmi
voi siete stanchi
loro sono veloci

## CAPITOLO 23: ORDINE DELLE PAROLE

# Ordine enfatico: Oggetto-Verbo-Soggetto

# Ordine con pronomi

# Ordine con avverbi

## CAPITOLO 24: PUNTEGGIATURA E PAUSE

# Virgola per elenchi

# Virgola per incisi

# Punto e virgola

# Due punti
