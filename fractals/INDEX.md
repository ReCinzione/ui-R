## Note d'Uso

1. I file sono nominati con il pattern: `ID_carattere_significato.svg`
2. Gli esagrammi usano ID numerici 00-63
3. I radicali extra usano prefisso `extra_`
4. Ogni SVG è 100x100px, scalabile
5. I colori sono semantici (rosso=energia, blu=acqua, verde=crescita, etc.)

## Integrazione con Prometeo

Questi radicali possono essere usati come:
- Glifi visivi per i frattali esistenti
- Base per nuovi frattali semantici
- Ponte tra italiano e cinese
- Sistema di pensiero visivo universale
